# Screenshots Folder

Place your campaign screenshots, landing page images, and performance data visualizations here.

## Recommended Structure:
- `case-study-1.png` - B2B Lead Generation campaign screenshots
- `case-study-2.png` - E-commerce ROAS campaign screenshots
- `google-ads-performance.png` - Google Ads dashboard screenshots
- `meta-ads-results.png` - Meta Ads performance screenshots
- `landing-page-*.png` - Landing page designs
- `email-templates-*.png` - Email marketing templates
- `seo-results-*.png` - SEO performance graphs

## Image Guidelines:
- Format: PNG or JPG
- Max width: 1920px
- Optimize images for web (compress before uploading)
- Use descriptive filenames
